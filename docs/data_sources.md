# Data sources

The GitHub repository contains key code only. Project-derived data supporting
the manuscript are not publicly released in this repository and will be made
available on request. Third-party products should be obtained from their
original providers.

## Project-derived data

Derived presence and background samples, model-ready environmental tables,
future-scenario prediction tables, area-summary tables, figure source data,
table source data, sensitivity-analysis outputs, quality-control summaries, and
selected geospatial evidence files are not included in this repository. Data
will be made available on request.

## Third-party products

- WorldClim climate layers: use the provider record and version cited in the
  manuscript.
- HydroRIVERS / HydroSHEDS hydrographic layers: use the HydroSHEDS and
  HydroRIVERS provider records and the cited Lehner and Grill reference.
- ESA WorldCover 2020: use the ESA WorldCover 10 m 2020 v100 record cited in
  the manuscript.
- Global Oases Distribution Dataset and its Cataloging System: use the Global
  Change Research Data Publishing & Repository record cited in the manuscript
  (Gui et al., 2025).

Do not redistribute original provider files unless the provider license allows
it and the final data record states the reuse conditions.
